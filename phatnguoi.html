Tra cứu phạt nguội — Demo

Nhập biển số, chọn loại xe, bấm "Tra cứu". (Dùng proxy để gọi API thực)
Mẫu: phatnguoixe.com
Kết quả sẽ hiển thị ở đây.
Ghi chú: file này là **frontend tĩnh**. Để tra cứu thực sự, bạn cần deploy 1 proxy server (Vercel/Netlify/VPS) và set biến BASE_PROXY trong script dưới thành domain proxy.

Thông tin kỹ thuật (ngắn)

  • DEFAULT: nếu không cấu hình proxy, trang sẽ hiển thị kết quả mẫu.
  • Để kết nối API thật: deploy serverless proxy và trả JSON theo dạng mẫu (xem phía dưới).
  • Trường JSON mẫu từ proxy (ví dụ): 
    {
  "status":"ok",
  "vehicle":"30A12345",
  "type":"Ô tô",
  "violations":[
    {"time":"2025-09-01 07:42","place":"Hà Nội - Nguyễn Trãi","code":"VĐ01","fine":"1.200.000","image_url": "..."},
    {"time":"2025-08-11 16:20","place":"HCM - Lê Duẩn","code":"VĐ02","fine":"800.000"}
  ]
}
